Comparação

É impossível comparar dois objetos jQuery

Todos os objeto jQuery são únicos. Mesmo que tenham sido criados do mesmo elemento DOM.

let a = $(“#logo”)

let b = $(“#logo”)

console.log(a === b) //false

O que fazer se a comparação for necessária?

Transforme os objetos jquery de volta nos respectivos elementos DOM (via método .get(index)) e em seguida os compare.

let c = a.get(0)

let d = b.get(0)

console.log(c === d) //true

Um get com índice negativo começa a contar do último elemento DOM.

console.log( $( "li" ).get( -1 ) );

O objeto jQuery é array like. Se você usar um subscrito (?) array, ele retorna os elementos DOM também.

let e = a[0]

let f = b[0]

console.log(e === f) //true
