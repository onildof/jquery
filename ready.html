$(function () {})

vulgo $(document).ready(function () {})

Uma página não pode ser manipulada seguramente até que o DOM (Document Object Model) esteja carregado.

.ready() é um object method que garante que o DOM completo foi carregado.

Tipicamente se usa .ready() com uma função anônima como parâmetro. Antigamente se escrevia assim:

$(document).ready(function () { … })

Mas desde o jquery 3.0, o objeto dessa função passou a poder ser qualquer coisa (ele simplesmente não importa mais). Por isso que agora prefere-se escrever:

$(function () { … }) (nem o próprio nome do método se escreve mais)

document do DOM

document é um elemento do DOM. Quando você envolve um elemento do DOM em “jquery parentheses”, recebe uma representação jquery daquele elemento DOM, ou seja, um objeto jquery mesmo.

.ready(), portanto é um object method do jquery, pois é chamado em um objeto jquery.

$(window).on("load")

O carregamento do DOM não inclui as imagens e outros elementos multimídia (apenas as tags). Se quiser aguardar o carregamento de todo o resto também, use:

$(window).on(“load”, function () { … })
